# Right Curly Bracket (`}`)

## `... }`: End Block
[→full documentation](https://mlochbaum.github.io/BQN/doc/block.html)

Starts a block, which starts with `}`. See [Begin Block](beginblock.md) for more details.

```bqn
   {𝕨 + 𝕩}   
(function block)

   {𝕨‿𝔽‿𝕩}   
(1-modifier block)

   {𝕨‿𝔽‿𝔾‿𝕩} 
(2-modifier block)

   {a ⇐ 5}   
{a⇐}

   {5+4+6}   
15
```
