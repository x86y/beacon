# Superscript Equals Sign (`⁼`)

## `𝔽⁼ 𝕩`, `𝕨 𝔽⁼ 𝕩`: Undo
[→full documentation](https://mlochbaum.github.io/BQN/doc/undo.html)

Invert the function `𝔽`, or use its defined inverse. Not all functions have inverses.


```bqn
   1 - 2
¯1

   1 -⁼ 2
¯1

   √ 16
4

   √⁼ 4
16

   ⋆ 1
2.718281828459045

   ⋆⁼ 2.718281828459045
1
```
