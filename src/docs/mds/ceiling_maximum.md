# Left Ceiling (`⌈`)

## `⌈ 𝕩`: Ceiling
[→full documentation](https://mlochbaum.github.io/BQN/doc/arithmetic.html#additional-arithmetic)

Round `𝕩` up to the nearest integer.

[Pervasive.](https://mlochbaum.github.io/BQN/doc/arithmetic.html#pervasion)

```bqn
   ⌈ 1.2‿π‿3‿7.89
⟨ 2 4 3 8 ⟩



```
## `𝕨 ⌈ 𝕩`: Maximum
[→full documentation](https://mlochbaum.github.io/BQN/doc/arithmetic.html#additional-arithmetic)

Maximum of `𝕨` and `𝕩`.

[Pervasive.](https://mlochbaum.github.io/BQN/doc/arithmetic.html#pervasion)

```bqn
   1 ⌈ 2
2

   2‿3 ⌈ 4‿2
⟨ 4 3 ⟩
```
